Комп'ютерний клуб

Учбовий ресурс для навчального гуртка для абітурієнтів спеціальності 123 "Комп'ютерна інженерія"
На цьому ресурсі будуть по мірі проведення зустрічей "Комп'ютерного клубу" додаватися корисні матеріали та додаткові джерела, які сприятимуть кращому розумінню розглянутих тем.
Орієнтовний план занять на найближчий час:
2024.10.14 Створення простої HTML-сторінки, публікація в мережі інтернет використовуючи засоби Github Pages.
2024.10.21 Знайомство з CMS WordPress.
2024.10.28 Основи стилізації вебсторінки. Знайомство з CSS.
2024.11.04 Створення анімації у вебдокументах та інші питання використання CSS.
Основи мови JavaScript.
JavaScript. Створення букмарклетів.
Отримання даних з інших ресурсів.
Деякі ресурси у контексті першої зустрічі, присвяченої створенню та публікації простої HTML-сторінки:
  • MDN - одне з основних довідкових джерел інформації щодо використання тегів і веб-розробки взагалі.
  • Стислий довідник (з використанням MDN) по тегам мови розмітки.
  • Типові приклади використання елементів HTML.
  • Сервіс для валідації (перевірки на відповідність стандартам) свого коду. Ним користувалися наприкінці заняття.
  • Сервіс віддаленого зберігання репозиторіїв. Завдяки можливості Github Pages ми і публікували свій ресурс у мережі інтернет.
  • Чудовий детальний розгляд того, що МОЖЕ бути в секції head вашого ресурсу, і для чого. Ми на занятті дуже спрощували це аспект)
